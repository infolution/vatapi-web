API documentation

Authentication

You need to obtain API key in your Account settings once you register. Just use it on every request as a parameter.

Output

Output is in JSON format. Output can be wrapped in a callback function for use with Javascript JSONP request.

API methods

Calc

GET
http://api.vatapi.co/calc/?country=UK&amount=1800&callback=myfunc&apikey=0NDiXLgGllnR7ziZL9bLDfIx0f1THacm

Method Calc is used for calculating tax amount based on customers country. It doesn't store tax amount for tax reports

query parameter description
country
required 		Country code (ISO 3166-1 alpha-2) for customer country. API will use this country tax rate to calculate tex amount
amount
required 		Amount in cents, don't use decimal amount, transform it to cents or other int value. Basically, you need to multiply it by 100. EXAMPLE: 25.54 USD becomes 2554 cents, and you submit cents
apikey
required 		Secret API key for authentication. You can obtain it in Account settings of VAT API website.
callback
optional 		Callback function that will wrap method result. It is used in Javascript with JSONP request

Sale

GET
http://api.vatapi.co/calc/?country=UK&amount=25000&callback=myfunc&apikey=6fpYAyHO0jwNTwoSi1p1L4D8ubVpECPg×tamp=1396855639

Method Sale is used for calculating tax amount based on customers country and storing it for later generation of quaterly tax reports. Then it is easy to file tax amounts for every country.

query parameter description
country
required 		Country code (ISO 3166-1 alpha-2) for customer country. API will use this country tax rate to calculate tex amount
amount
required 		Amount in cents, don't use decimal amount, transform it to cents or other int value. Basically, you need to divide it by 100. EXAMPLE: 25.54 USD becomes 2554 cents, and you submit cents
apikey
required 		Secret API key for authentication. You can obtain it in Account settings of VAT API website.
timestamp
required 		Unix timestamp value. It represents time when transaction was done.
callback
optional 		Callback function that will wrap method result. It is used in Javascript with JSONP request

Javascript


        var amountcents = 25.54 * 100; //Transform into cents
        var data = {country: 'UK',
                    amount: amountcents,
                    apikey: '6fpYAyHO0jwNTwoSi1p1L4D8ubVpECPg'
        };

        $.ajax({
            url: 'http://api.vatapi.co/calc/',
            data: data,
            success: function (res) {
                console.log('Response received. ' + res);
                //Do something with the response
            },
            dataType: "jsonp",
            jsonp: "callback"
        });
